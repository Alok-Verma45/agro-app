# Agro App 🌱

## 📌 Overview
Agro App is a scalable system designed for agricultural businesses like fertilizer and pesticide distributors.

It helps manage:
- Inventory (Stock)
- Udhaar (Credit)
- Billing
- Staff operations

Future goal:
- Customer ordering system (like mini e-commerce)

---

## 🏗️ Project Architecture

This project follows a **modular structure**:

agro-app/
├── agro-app-server/   # Backend (Spring Boot)
├── agro-app-client/   # Frontend (React - upcoming)
├── docs/              # Documentation
├── scripts/           # DB / setup scripts


---

## ⚙️ Setup Instructions

### 1. Clone the repository
git clone https://github.com/your-username/agro-app.git

### 2. Backend Setup
- Navigate to `agro-app-server`
- Configure `application.yml`
- Create PostgreSQL DB: `agro_app`
- Run Spring Boot application

### 3. Frontend Setup (Future)
- Navigate to `agro-app-client`
- Install dependencies
- Start React app
